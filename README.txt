The project is a simple Crud using JPA. 
All the layers are separetade in diffrent package's. 

In this project there is only one entity called Product. 
The entity is used in the DAO package where the 
communication with the database is present.
The layer for "business logic" is in the so called menuswitch 
package where the class for communication between the user interface and 
the DAO is present.

The layer for user interface is in the package ui. All the menu text is presented in the console using 
the method System.out.println().
